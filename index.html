API Endpoints

Access the following JSON endpoints to query the NIST CMVP data:

./api/index.json
API information and available endpoints
./api/metadata.json
Metadata about the dataset (last update, total count, etc.)
./api/modules.json
Complete list of all validated modules

Quick Start

Try these examples to get started:

Using curl:

curl ./api/metadata.json

Using JavaScript:

fetch('./api/modules.json') .then(r => r.json()) .then(data => console.log(data));
